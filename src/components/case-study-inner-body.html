The Logic.

We implemented a caching layer using Redis to serve static financial data, reducing database hits by 95%. Here is the core logic for the data fetcher:

fetchData.js ● Node.js 
const getData = async (userId) => {
  // Check Cache First
  const cachedData = await redis.get(`user:${userId}`);
  
  if (cachedData) {
    console.log('Returning cached payload');
    return JSON.parse(cachedData);
  }

  // Fallback to DB (Only 5% of requests)
  const dbData = await database.query(userId);
  await redis.set(`user:${userId}`, JSON.stringify(dbData), 'EX', 60);
  
  return dbData;
}
